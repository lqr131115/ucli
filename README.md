## 一个集成的脚手架 :green_heart:

- 一个支持`ejs`动态渲染的**模板下载器**
- 一个支持`Github | Gitee`双平台的**源码下载器**
- 一个支持代码自动化`校验`和`测试`**代码检查器**
- 一个支持代码`自动提交`的**代码提交器**
